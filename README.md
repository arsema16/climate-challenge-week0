# African Climate Trend Analysis - Week 0

## Project Overview
This repository contains exploratory data analysis of historical climate data for Ethiopia, Kenya, Sudan, Tanzania, and Nigeria (2015–2026) to inform Ethiopia's position for COP32.

## Setup Instructions
1. **Clone the repo:**
   `git clone https://github.com/arsema16/climate-challenge-week0.git`
2. **Create a virtual environment:**
   `python -m venv venv`
3. **Activate the environment:**
   - Windows: `.\venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
4. **Install dependencies:**
   `pip install -r requirements.txt`
